SwiftChess 
=============

*For iOS*. Written in Swift.


---


![Layout](http://www.agmotif.com/blog/wp-content/uploads/2016/04/ChessBoard.png)


**Features**

* Playable chess Game
* Legal move detection for pieces
* Check detection
* Checkmate detection
* Pawn promotion
* Special moves: En passant, Castling
* Included Unit Test and UI Tests
